
# Solvara Safaris

An ultra-luxury safari concierge web application targeting ultra-high net worth 
individuals, family offices, and personal managers. Built as a university web 
development group project.

**Live site:** https://solvara-xi.vercel.app  
**GitHub:** https://github.com/mahakidnani/solvara

---

## Project Overview

Solvara Safaris is not a standard travel booking site. It is designed as a 
private digital estate — the online equivalent of a bespoke concierge service. 
The site covers twelve African safari destinations and provides end-to-end 
journey curation from doorstep collection to post-trip memory book delivery.

The project began as an earlier prototype called WildVerse (built by a team 
member), which had strong technical foundations but no defined target audience 
or brand direction. After structured design thinking sessions and competitor 
analysis — referencing Singita, Abercrombie & Kent, and &Beyond — the team 
pivoted to a fully redefined concept with a clear UHNWI audience, dark 
editorial aesthetic, and the Solvara brand identity.

## Pages

| Route | Page | Description |
|---|---|---|
| `/` | Homepage | Hero, personalisation gateway, destinations strip, services, membership |
| `/expeditions` | Expedition Collection | Filterable safari packages by destination and type |
| `/destinations` | The Atlas | 12 destination overview cards |
| `/destinations/:id` | Destination Detail | Per-destination Virtual Verse, lodges, experiences, wildlife calendar |
| `/wildlife` | Wildlife Guide | Species grid with filtering |
| `/private-jets` | Private Jets | Fleet showcase and aviation enquiry |
| `/accommodation` | Accommodation | Lodge and villa collection |
| `/conservation` | Conservation | Pledge tracker and partner organisations |
| `/membership` | Solvara Private | Three-tier membership — Pathfinder, Dynasty, Sovereign |
| `/concierge` | Concierge | 4-step enquiry wizard |
| `/dashboard` | Private Portal | Client journeys, conservation impact, loyalty status |
| `/credits` | Credits | Full photography and asset credits |

## Design System

**Colour palette:**

| Token | Hex | Usage |
|---|---|---|
| Ink Black | `#0F0E0B` | Primary background |
| Charcoal | `#1A1815` | Secondary sections |
| Forest Green | `#3B4A3F` | Rich accent sections |
| Plum | `#4A2D35` | Membership section |
| Inky Blue | `#3A4455` | CTA sections |
| Gold | `#C9A84C` | All accents, CTAs, highlights |
| Warm Cream | `#E8DFCC` | Primary text |
| Mushroom Taupe | `#B5A48A` | Body copy, muted text |

- Display / Hero: Cormorant Garamond (300–500, italic)
- Titles / Navigation: Cinzel (400–500)
- Body copy: EB Garamond (400–500)
- UI labels / Buttons: Lato (200–400)
